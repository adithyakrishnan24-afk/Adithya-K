# Adithya K — Portfolio

A personal portfolio website with animated transitions and project showcase.

## Files
- `index.html` — Main portfolio card (home page)
- `transition.html` — Animated loading screen between portfolio and projects
- `project.html` — Projects showcase page
- `age.html` + `age-script.js` + `age-style.css` — Age Calculator project
- `style.css` — Portfolio card styles
- `project-style.css` — Projects page styles

## Deploy to GitHub Pages
1. Create a new GitHub repo (e.g. `Adithya-K`)
2. Push all files to the `main` branch
3. Go to repo Settings → Pages → Source: Deploy from branch → `main` / `root`
4. Your portfolio will be live at: `https://<your-username>.github.io/Adithya-K/`

## Update GitHub links
In `project.html`, update the GitHub button URLs to match your actual repos:
- `https://github.com/YOUR_USERNAME/dyscalculia`
- `https://github.com/YOUR_USERNAME/ai-stock-platform`
